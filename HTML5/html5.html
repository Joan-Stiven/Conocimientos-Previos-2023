HTML5

Etiquetas semánticas:

  • <header>: Define una cabecera para un documento o una sección.
  • <nav>: Define un conjunto de enlaces de navegación.
  • <article>: Define un contenido independiente que podría estar distribuido y reutilizable.
  • <section>: Define una sección en un documento.
  • <footer>: Define un pie de página para un documento o una sección.
  • <aside>: Define contenido que debería ser apartado del contenido principal.

Etiquetas de texto:

  • <p>: Define un párrafo.
  • <h1> to <h6>: Define encabezados de diferentes niveles.
  • <a>: Define un enlace.
  • <strong>: Define texto fuerte, importante.
  • <em>: Define texto enfatizado.

Etiquetas de agrupación:

  • <div>: Define una división o una sección en un documento.
  • <span>: Define una pequeña sección de texto.

Etiquetas multimedia:

  • <img>: Define una imagen.
  • <audio>: Define contenido de audio.
  • <video>: Define contenido de video.

Etiquetas de tablas:

  • <table>: Define una tabla.
  • <tr>: Define una fila en una tabla.
  • <th>: Define un encabezado de celda en una tabla.
  • <td>: Define una celda en una tabla.

Etiquetas de formularios:

  • <form>: Define un formulario.
  • <input>: Define un campo de entrada en un formulario.
  • <select>: Define una lista desplegable.
  • <textarea>: Define un área de texto multilinea.
