SQL

Temática vista en SQL

SQL es un lenguaje de programación utilizado para administrar y manipular bases de datos relacionales. Aquí hay algunos conceptos básicos:

  • Bases de datos relacionales: Sistemas de gestión de bases de datos que almacenan datos en tablas relacionadas.
  • Tablas: Estructuras que contienen datos organizados en filas y columnas.
  • Consultas (Queries): Instrucciones SQL utilizadas para recuperar datos de una base de datos.
  • INSERT, UPDATE, DELETE: Instrucciones utilizadas para insertar, actualizar o eliminar datos de una tabla.
  • SELECT: Instrucción utilizada para recuperar datos de una o más tablas.
  • WHERE: Cláusula utilizada para filtrar los resultados de una consulta.
  • JOIN: Cláusula utilizada para combinar datos de dos o más tablas relacionadas.
  • Índices: Estructuras utilizadas para mejorar la velocidad de recuperación de datos en una base de datos.
  • Transacciones: Conjunto de operaciones que se ejecutan como una unidad atómica e indivisible.
  • Relaciones: Vínculos establecidos entre las filas de diferentes tablas utilizando claves primarias y foráneas.
  • Integridad referencial: Restricciones que garantizan la consistencia y validez de los datos en una base de datos.
