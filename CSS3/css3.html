CSS3

Hojas de estilo en cascada CSS 3

CSS3 es la última versión del lenguaje de hojas de estilo en cascada utilizado para dar estilo a páginas web. Proporciona una amplia variedad de funciones y propiedades para mejorar la apariencia y el diseño de un sitio web.

Tipos de selectores

Los tipos de selectores en CSS3 incluyen:

  • Selector de tipo: Selecciona elementos basados en su nombre de etiqueta.
  • Selector de clase: Selecciona elementos basados en su clase.
  • Selector de ID: Selecciona un elemento basado en su ID único.
  • Selector descendente: Selecciona elementos anidados dentro de otros elementos.
  • Selector de atributo: Selecciona elementos basados en sus atributos.
  • Selector de pseudoclase: Selecciona elementos en diferentes estados.
  • Selector de pseudoelemento: Selecciona partes específicas de un elemento.
  • Selector de grupo: Agrupa varios selectores para aplicarles el mismo estilo.
  • Selector universal: Selecciona todos los elementos en un documento.
  • Selector child: Selecciona elementos que son hijos directos de otro elemento.

Pseudo clases

Las pseudo-clases en CSS3 permiten aplicar estilos a elementos en estados específicos, como cuando un enlace es activado, se encuentra en el estado de foco, o al pasar el cursor sobre él.

  • :hover: Cambia el estilo cuando el cursor se sitúa sobre el elemento.
  • :active: Cambia el estilo cuando el elemento está activo.
  • :focus: Cambia el estilo cuando el elemento obtiene el foco.
  • :first-child: Selecciona el primer hijo de un elemento.
  • :nth-child(n): Selecciona el enésimo hijo de un elemento.
  • :nth-of-type(n): Selecciona el enésimo hijo del mismo tipo de un elemento.
  • :not(selector): Selecciona elementos que no cumplen con el selector especificado.
  • :last-child: Selecciona el último hijo de un elemento.
  • :nth-last-child(n): Selecciona el enésimo hijo desde el final de un elemento.
  • :nth-last-of-type(n): Selecciona el enésimo hijo desde el final del mismo tipo de un elemento.

Unidades de medida relativas

Las unidades de medida relativas en CSS3 permiten definir tamaños y distancias en relación con otras propiedades, como el tamaño de fuente del elemento padre o el tamaño de la ventana del navegador.

  • em: Relativo al tamaño de fuente del elemento.
  • rem: Relativo al tamaño de fuente del elemento raíz (html).
  • %: Porcentaje del tamaño del elemento padre.
  • vw: 1% del ancho de la ventana.
  • vh: 1% de la altura de la ventana.

Fondos y bordes

Las propiedades de fondo y borde en CSS3 permiten estilizar elementos con colores, imágenes, sombras y bordes personalizados. Esto incluye la definición de colores de fondo, imágenes de fondo, estilos de borde, radios de borde y sombras.

  • background-color: Define el color de fondo.
  • background-image: Establece una imagen como fondo.
  • border: Define el estilo, ancho y color del borde.
  • border-radius: Define la curvatura de las esquinas.
  • box-shadow: Añade sombras a los elementos.
  • outline: Establece una línea alrededor de un elemento sin afectar el diseño.

Modelo de cajas

El modelo de cajas en CSS describe la estructura de un elemento en términos de su contenido, padding, borde y margen. Esto permite controlar el diseño y el espacio que ocupa un elemento en una página web.

  • width: Ancho del contenido de un elemento.
  • height: Altura del contenido de un elemento.
  • margin: Espacio externo alrededor de un elemento.
  • padding: Espacio interno alrededor del contenido de un elemento.
  • border-box: Modelo de caja que incluye el ancho y alto del contenido y el padding, pero no el margen.
  • content-box: Modelo de caja que incluye solo el ancho y alto del contenido.

Position

La propiedad CSS position controla el método de posicionamiento de un elemento en una página web. Los valores posibles incluyen static, relative, absolute, fixed y sticky.

Display

La propiedad CSS display controla el método de visualización de un elemento en una página web. Los valores posibles incluyen block, inline, inline-block, flex, grid y otros.

Responsive Design "Media Queries"

El diseño responsivo en CSS utiliza Media Queries para adaptar el diseño de una página web a diferentes dispositivos y tamaños de pantalla. Esto permite que el sitio se vea bien y sea funcional en dispositivos móviles, tablets y computadoras de escritorio.
